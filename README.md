# elian-agent
This is a repository of complex agent algorithms for daily learning and development. The agent implementations in this repository reference the core design patterns of Claude Code ,OpenClaw and Hermes Agent, and use Python to quickly verify the agent design ideas.

---

## project struct

- elian_agent_cc是claude code的核心agent算法
- elian_agent_oc是openclaw的核心agent算法
- elian_agent_hermes是hermes agent的核心agent算法
- elian_agent是三者抽象出来的可复用的agent算法
- go是elian_agent的go语言版本
- ts是elian_agent的typescript版本
